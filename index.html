Comandos necessários para criar um Site

Comandos principais para se criar um site:

Os principais comandos nos quais você deve utilizar, são:

-> "DOCTYPE HTML": Essa TAG define que o documento é em HTML.

-> "HTML": Esta TAG é a principal de uma página HTML.

-> "HEAD" Essa é a TAG que abre o bloco de informações sobre o site.

-> "TITLE": Aqui colocamos o título que daremos ao nosso site.

Agora que mexemos com a parte "head", vamos mexer com a parte do "body":



Os principais comandos utilizados na parte body do site, são:

Aliás, antes de dizer isso, "body" é a parte do site na qual ficará seus textos e títulos.



    "h1": Esta TAG define um título de nível 1. Esta TAG pode ser estendida até o nível 6.

    "p": Esse comando simboliz o parágrafo que escreveremos, igual esse no qual você está lendo agora "HAHAHA".

    "a": Esta TAG é muito legal, ela adiciona um link ao texto colocado entre este comando. As pessoas normalmente chamam isso de "Hyperlink"

    "br" Esta TAG serve para pular algo referente a uma linha. Sabe quando você está escrevendo algo e oula uma linha? É tipo isso.


A segunda parte, o "Style.CSS:"



Alguns dos comandos nos quais podemos utilizar no "Style.CSS" são:

    "linear-gradient": Este comando permite definir um gradiente linear no fundo do site.

    "radial-gradient": Permite definir um gradiente radial de background.

    "rgb": Este comando permite definir cores através do sistema RGB com valor entre 0 a 255.

    "id": Este é um comando no qual permite que você personalize somente uma identificação do site.

    "class": Este comando permite que você personalize mais de uma identificação de uma vez.

    "bacground-color": Este comando serve para definir os valores de fundo individuais em um único lugar na folha de estilos.
